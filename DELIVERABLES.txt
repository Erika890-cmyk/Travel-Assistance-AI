================================================================================
                        HOW TO USE THE SYSTEM
================================================================================

INSTALLATION (already completed):
1. Python 3.13 environment set up
2. Virtual environment created and activated (.venv/)
3. All dependencies installed from requirements.txt
4. Project directory: /vercel/share/v0-project

RUNNING THE SYSTEM:

Step 1: Set OpenAI API Key
    export OPENAI_API_KEY="your-api-key-here"

Step 2: Start Streamlit Application
    cd /vercel/share/v0-project
    source .venv/bin/activate
    streamlit run app.py

Step 3: Open Browser
    http://localhost:8501

Step 4: Use the Interface
    - Enter source city (e.g., Delhi)
    - Enter destination city (e.g., Goa)
    - Set number of travelers
    - Set budget
    - Click "Plan My Trip"
    - View results in tabs

TESTING (without web UI):
    python test_cli.py
